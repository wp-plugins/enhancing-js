== Description ==

Enhancing JS plugin will enable to edit external JavaScript.
This JavaScript is output through WordPress rewrite API.
In addition, this plugin implements simple visual editor for JavaScript.

If you will install this plugin with ['Enhancing CSS' plugin](http://wordpress.org/extend/plugins/enhancing-css/),
You can customize themes without PHP programming!

* [Plugin Homepage](http://firegoby.theta.ne.jp/wp/enhancingjs)
* [Support](http://wordpress.org/tags/enhancing-js)

= Some features: =

* You can edit External JavaScript on visual editor.
* You can manage WordPress Default JavaScript frameworks.
* Conditional GET and ETag support.

= Translators: =

* Japanese(ja) - [Takayuki Miyauchi](http://firegoby.theta.ne.jp/)

You can send your own language pack to me.


== Installation ==

* Upload the "Enhancing JS" plugin to your blog.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Visit "Enhancing JS" in Appearance.

== Frequently Asked Questions ==

= This JavaScript is physical file? =
No. This JavaScript output through WordPress rewrite API.

= JavaScript path? =
Example: http://your_site_url/EnhancingJS.js

= Can I use "Enhancing JS" with prototype.js, jquery and other JavaScript frameworks? =
Yes.
Please check, if you want to load WordPress default javascripts on Admin Panel.

= What are Capabilities? =
edit_theme_options
